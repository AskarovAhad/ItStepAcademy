Инструкция использования проводника!
1)Для работы с файлом: команда + /f
2)ля работы с папками/директориями: команда + /d

Комманды
cd - перемещает нас в папку
cd.. - возвращает в главную директорию
dir - перемещает нас в введенную директорию
dir /s /f, dir /s /d - чтоы узнать размер папки/файла
del /f, /d  -  удаляет файл/папку, нужно указать полный путь удаления папки
create /f, /d - создает папку/файл
copy /f, /d - копирует все содержимое папки или файл (если в папке будут файлы с одинаковым именем, программа выдаст ошибку)
move /f, /d - перемещает все содержимое папки или файл (если в папке будут файлы с одинаковым именем, программа выдаст ошибку)
rename /f, /d - переименовывает папку/файл
manual - открывает инструкцию (этот файл)
space - выводит общее/доступное/свободное пространство на диске D и C в мегабайтах
exit - завершает работу программы
log - открывает папку с логами программы
